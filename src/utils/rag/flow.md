# graphRAG流程

节点:
- 文件夹
- 文件
- 类
- 函数

边:
- 文件夹包含文件夹
- 文件夹包含文件(最大代码块)
- 代码块AST关系(多种多层 TODO)

## 流程图

```mermaid
graph TD;
    A[git拉取/zip上传] 
    -->B[获取代码]-->B1[遍历文件文件夹]
    --> C{类型?};
    C --> |文件夹| D[图插入节点
    文件夹]--> B1;
    C --> |文件| E[
        图插入节点
        文件
        ]

    E --> 文件与文件夹关系 --> 文件夹与文件夹关系
    E --> 构建ast --> 代码块从属当前文件关系 --> 代码块与代码块关系
```
